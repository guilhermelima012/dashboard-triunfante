# Dashboard Comercial - Triunfante

Projeto desenvolvido em Python utilizando Streamlit para análise de vendas.

## Funcionalidades

- KPIs de receita, quantidade e ticket médio
- Filtros por data, região, categoria, cliente e marca
- Análise de vendas por categoria
- Top produtos
- Evolução mensal de vendas
- Tabela detalhada dos dados

## Tecnologias

- Python
- Streamlit
- Pandas
- Plotly
